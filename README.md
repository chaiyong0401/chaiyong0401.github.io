# Chaiyong Mun - Academic Website

This is a lightweight static academic homepage built from the provided CV.

## Files

- `index.html` - main website
- `styles.css` - responsive styling
- `script.js` - small footer year script
- `assets/Chaiyong_Mun_CV.pdf` - downloadable CV

## GitHub Pages deployment

1. Create a GitHub repository, for example `chaiyong0401.github.io` or `Chaiyong-Mun`.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to Settings > Pages.
4. Select the branch to deploy from, usually `main`, and root folder `/`.
5. Save. GitHub Pages will provide the public URL.

## Customization

Replace the `CM` placeholder in the profile card with a profile photo by adding an image to `assets/` and editing the `profile-card` section in `index.html`.
